  \section{实验结果分析}
  本次实验选择了数据集中的开盘价、前收盘价、总市值和流通市值作为特征来预测涨跌情况。
  因为数据量较少，因此采用了随机K折交叉验证的方法对样本进行了交叉训练和验证。
  从结果中可以看出，F1-Score平均在0.5左右，说明模型预测的准确率并不高，
  但是也能在一定程度上进行预测。

  \newpage
  \paragraph*{有两个原因可能会导致这种结果}
  \begin{enumerate}
    \item 样本特征的选择与结果相关性差，模型无法从数据中学习到特征
    \item 数据集较少，不足以反映出数据的特征
  \end{enumerate}
